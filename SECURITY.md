# Security Policy

## Reporting a vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Email: priteshjha2711@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive a response within 72 hours.

## Scope

This policy covers the `anonypii` Python package and its bundled model download tooling.
It does not cover the underlying HuggingFace model weights, which are governed by their
respective licenses (Apache 2.0).
